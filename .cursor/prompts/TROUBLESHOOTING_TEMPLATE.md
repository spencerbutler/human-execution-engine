# Cursor Wrapper: Universal CI Troubleshooting Guide Template

**Canonical File**: `prompts/TROUBLESHOOTING_TEMPLATE.md`

This is a Cursor IDE wrapper that points to the canonical troubleshooting guide template. The actual content is maintained in the canonical file above.

**Purpose**: Provides a template version of the CI/CD troubleshooting guide with template variables for cross-project use.

**Last Updated**: 2026-01-24

**Usage**: This wrapper ensures Cursor IDE integration while maintaining the canonical template in the prompts/ directory as required by HEE rules.

**Template Variables**: Replace all `{VARIABLE}` placeholders with project-specific values before use.
